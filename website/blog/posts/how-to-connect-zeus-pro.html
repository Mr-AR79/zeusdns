آموزش کامل اتصال به زئوس پرو در ۵ دقیقه!

اندروید

اتصال از طریق اپلیکیشن زئوس:

  1. نرم افزار اختصاصی زئوس برای اندروید را به صورت مستقیم دریافت و نصب نمایید.
  2. نرم افزار را اجرا کرده و اجازه دسترسی ها را بدهید.
  3. مطابق این تصویر توکن خود را وارد و ثبت کنید.
  4. از بخش سرورهای دی ان اس، سرور پرو را انتخاب کنید.
  5. دکمه اتصال را بفشارید.

اتصال از طریق تنظیمات دستی:

قبل از تنظیم دی ان اس، از طریق این لینک توکن خود را ثبت کنید و سپس مراحل زیر را انجام دهید.

در اندروید ۹ و بالاتر

  1. وارد تنظیمات شوید.
  2. به مسیر Network & Internet > Advanced > Private DNS بروید.
  3. گزینه Private DNS provider hostname را انتخاب کنید.
  4. عبارت زیر را وارد کنید:
    pro.zeusdns.ir
  5. روی Save بزنید.

در اندروید پایین تر از ۹ و در صورت اتصال به Wi-Fi

  1. وارد تنظیمات گوشی شوید (Settings).
  2. بخش اتصالات یا Network & Internet را باز کنید.
  3. وارد قسمت Wi-Fi شوید و شبکه متصل را انتخاب کنید.
  4. گزینه‌ی Modify Network یا Advanced را انتخاب کنید.
  5. در قسمت IP settings، حالت را به Static تغییر دهید.
  6. در بخش DNS مقادیر زیر را وارد کنید:
    • DNS 1: 37.32.5.34
    • DNS 2: 37.32.5.35
  7. تغییرات را ذخیره کنید.

توجه: این تنظیم فقط برای همان شبکه Wi-Fi اعمال می‌شود. در صورت تغییر شبکه، باید تنظیمات را مجدد انجام دهید.

ویندوز

اتصال از طریق اپلیکیشن زئوس:

  1. نرم افزار اختصاصی زئوس برای ویندوز را به صورت مستقیم دریافت و نصب نمایید.
  2. طبق تصویر زیر میتوانید از این قسمت، آدرس های Dns pro را وارد کنید.
  3. سپس از طریق این بخش توکن خود را وارد کرده و مدت زمان بررسی خودکار آی پی برای ثبت توکن را به دلخواه ویرایش کنید، و در نهایت دکمه استارت را بزنید.

اتصال از طریق تنظیمات دستی:

قبل از تنظیم دی ان اس، از طریق این لینک توکن خود را ثبت کنید و سپس مراحل زیر را انجام دهید.

در اندروید ۹ و بالاتر

  1. وارد Control Panel شوید.
  2. به مسیر زیر بروید:
    Network and Internet > Network and Sharing Center > Change adapter settings
  3. روی شبکه (Wi-Fi یا Ethernet) که به آن متصل هستید راست‌کلیک کرده و گزینه Properties را انتخاب کنید.
  4. گزینه Internet Protocol Version 4 (TCP/IPv4) را انتخاب کرده و روی Properties بزنید.
    گزینه Use the following DNS server addresses را فعال کنید.
  5. مقادیر زیر را وارد کنید:
    • Preferred DNS server: 37.32.5.34
    • Alternate DNS server: 37.32.5.35
  6. روی OK بزنید و تمام پنجره‌ها را ببندید.
  7. در صورت لزوم مرورگر خود را یکبار بسته و مجدد باز کنید.

سیاست حفظ حریم خصوصی

  • هیچ‌کدام از درخواست‌های DNS شما (Queries) ثبت یا ذخیره نمی‌شود.
  • آدرس IP، زمان اتصال یا محتوای ترافیک شما تحلیل نمی‌شود.
  • ما از پروتکل‌های رمزنگاری‌شده برای انتقال DNS استفاده می‌کنیم.
  • اطلاعات شخصی شما (در صورت تماس) فقط برای پشتیبانی استفاده می‌شود و حذف می‌گردد.
  • ما با هیچ نهاد یا شرکتی اطلاعات کاربران را به اشتراک نمی‌گذاریم.

حریم خصوصی شما، اولویت اول ماست. بدون لاگ، بدون ردیابی.
